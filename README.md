#  SchoolCore

Sistema de gerenciamento escolar desenvolvido em Python com integração ao banco de dados.

## Sobre:

Desenvolvi esse sistema como projeto acadêmico para praticar Python orientado a objetos e integração com banco de dados NoSQL.

O SchoolCore roda inteiramente no terminal e permite gerenciar os principais elementos de uma escola: alunos, professores, turmas, disciplinas e notas.

##  Funcionalidades:

- Cadastro e gerenciamento de alunos
- Cadastro e gerenciamento de professores
- Criação e organização de turmas e disciplinas
- Lançamento e consulta de notas
- Persistência de dados com MongoDB

##  Tecnologias:

- Python 3.x
- MongoDB (local)
- PyMongo
